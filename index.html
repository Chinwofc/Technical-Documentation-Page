Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language.

What you should already know

This guide assumes you have the following basic background:

  • A basic understanding of HTML and CSS.
  • Some understanding of programming concepts such as variables, control structures, functions, etc.
JavaScript and Java

Despite the name, JavaScript and Java have very little in common. They are both programming languages but are used for different purposes.

In JavaScript, variables are used to store data values. You can declare variables using var, let, or const.

Operators
